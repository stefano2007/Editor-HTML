# Editor-HTML
Editor HTML Desktop Windows em C# com Web Browser

* Criar arquivos HTML;
* Inserir imagens;
* Formatar texto;
* Colar conteúdos já formatados por exemplo Word ou Paginas WEB;
* Imprimir HTML;
* Salvar/Exportar HTML.
